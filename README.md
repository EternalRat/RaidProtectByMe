# RaidProtectByMe

Fully secure bot for Discord servers

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

What things you need to install the software and how to install them

```
npm
nodejs
via npm :
        - discordjs@11
        - nodemon
        - fs
        - moment
        - common-tags
```

### Installing


Cloning the repository

```
git clone https://github.com/EternalRat/RaidProtectByMe.git
```

And then install nodejs and npm, all requirements are already includes except those one

## Deployment

Type this on a console, then the bot will launch by himself
```
npm run dev
```
